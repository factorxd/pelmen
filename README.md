# 🥟 Пельмень — Простой и вкусный шаблонизатор документов

**Пельмень** — это десктопное приложение для Windows, которое превращает скучное заполнение документов в быстрое и приятное занятие. Создайте шаблон в Word, настройте поля в программе и получайте готовые документы за секунды.

## ✨ Основные возможности

*   **Массовая генерация**: Создайте сотни документов из одного шаблона, загрузив данные из Excel или CSV.
*   **Умные шаблоны**: Используйте условия (`if`) и циклы (`for`) прямо в Word.
*   **Удобная настройка**: Дайте полям русские имена, сгруппируйте их по категориям и настройте типы (текст, число, дата, да/нет).
*   **Автосохранение**: Заполняйте форму в своём темпе — всё сохранится автоматически.
*   **Предпросмотр**: Посмотрите, как будет выглядеть документ, прямо в Microsoft Word.

## 🚀 Быстрый старт (для пользователей)

1.  Скачайте последнюю версию из раздела [Releases](https://github.com/factorxd/pelmen/releases).
2.  Распакуйте архив в любую папку.
3.  Запустите файл `Пельмень.exe`.
4.  При первом запуске укажите папку с вашими шаблонами `.docx`.
