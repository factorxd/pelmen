# 🥟 Пельмень — простой шаблонизатор документов

**Пельмень** — это десктопное приложение для Windows, которое позволяет создавать документы из шаблонов Word, подставляя данные из удобной формы. Идеально для договоров, накладных, сертификатов и любых повторяющихся документов.

![Версия](https://img.shields.io/badge/version-1.0-blue) ![Python](https://img.shields.io/badge/python-3.13-green) ![PySide6](https://img.shields.io/badge/PySide6-6.11-orange)

## ✨ Возможности

- **Шаблоны в Word** — используйте теги `{{ поле }}`, циклы `{% for ... %}`, условия `{% if ... %}`.
- **Удобная форма** — поля группируются по категориям в виде вкладок.
- **Динамические блоки** — добавляйте/удаляйте строки (например, список товаров).
- **Массовая генерация** — создавайте сотни документов из CSV/Excel.
- **Предпросмотр** — открывает сгенерированный документ в Word.
- **Автосохранение черновиков** — данные не потеряются при случайном закрытии.
- **Категории полей** — организуйте поля по смыслу.
- **Пресеты** — сохраняйте часто используемые значения (ФИО, адреса, суммы) и вставляйте в любое поле через контекстное меню.
- **Тёмная тема** — комфортная работа в любое время суток.
- **Поиск шаблонов** — быстрая фильтрация в дереве файлов.
- **Горячие клавиши** — ускоряйте работу.

## 🚀 Быстрый старт

1. **Скачайте последнюю версию** из раздела [Releases](https://github.com/factorxd/pelmen/releases).
2. **Распакуйте архив** в любую папку.
3. **Запустите** `Пельмень.exe`.
4. **Выберите папку** с вашими шаблонами `.docx` (при первом запуске).
5. **Кликните на шаблон** — справа появится форма.
6. **Заполните поля**, добавьте строки при необходимости.
7. **Нажмите «Скачать DOCX»** — готовый документ сохранится на диск.

## 📝 Создание шаблона в Word

1. Откройте Word, создайте документ.
2. В места, куда нужно подставить данные, вставьте теги вида `{{ nazvanie_polya }}`.  
   **Только латиница!** Пример: `{{ client_name }}`, `{{ date }}`.
3. Для повторяющихся блоков (списки) используйте конструкцию:
   ```jinja
   {% for item in spisok %}
   {{ item.nazvanie }}, {{ item.cena }}
   {% endfor %}
   ```
4. Для условий (показать/скрыть фрагмент) — `{% if flag %}...{% endif %}`.  
   Флаги управляются чекбоксами в форме (тип поля `bool`).
5. Сохраните файл как `.docx`.

> **Важно:** имена переменных только латиница, без пробелов. Кириллица не поддерживается движком Jinja2.

## ⚙️ Настройка шаблона в программе

После выбора шаблона нажмите **«Настроить отображаемые имена и категории»**.  
Вы сможете:

- Задать русские подписи для полей (например, `client_name` → «Имя клиента»).
- Выбрать тип поля: `text`, `number`, `date`, `bool` (чекбокс).
- Указать формат даты или числа (например, `DD.MM.YYYY`, `1 234,56 ₽`).
- Создать категории и сгруппировать поля по вкладкам.

## 🧩 Пресеты (часто используемые значения)

- Откройте **«Файл → Пресеты»**, добавьте новые значения (например, «ООО Ромашка» → «1234567890»).
- В любом поле формы (текст, число, дата) щёлкните **правой кнопкой мыши** → «Вставить из пресетов» → выберите нужный.
- Пресеты группируются по категориям, которые вы зададите при добавлении.

## 📁 Массовая генерация из CSV/Excel

1. Подготовьте файл с колонками, имена которых совпадают с полями шаблона.
2. В программе выберите шаблон, нажмите **«Файл → Массовая генерация из CSV/Excel»**.
3. Загрузите файл, сопоставьте колонки с полями.
4. Нажмите **«Сгенерировать ZIP»** — получите архив с документами для каждой строки.

## 🛠️ Для разработчиков

### Запуск из исходников

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/factorxd/pelmen.git
   cd pelmen
   ```
2. Создайте виртуальное окружение и установите зависимости:
   ```bash
   python -m venv venv
   venv\Scripts\activate   # Windows
   pip install -r requirements.txt
   ```
3. Запустите приложение:
   ```bash
   python main.py
   ```

### Сборка `.exe` через PyInstaller

```bash
pyinstaller --onedir --windowed --name "Пельмень" --icon icons/pelmen2.ico --add-data "icons;icons" --add-data "data;data" main.py
```

(подробности в документации PyInstaller)
