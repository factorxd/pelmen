🥟 Пельмень – руководство пользователя

Программа позволяет создавать документы из шаблонов Word, автоматически подставляя данные из формы.

📁 1. Выбор папки с шаблонами

При первом запуске укажите папку, где лежат ваши шаблоны (.docx). Программа покажет их в левой панели в виде дерева.
Файл → Выбрать папку с шаблонами – изменить папку.

📝 2. Создание шаблона в Word

В Word вставьте теги вида {{ nazvanie_polya }} (только латиница!).
Пример: Договор с {{ client_name }} от {{ date }}.

Для повторяющихся блоков (списки) используйте конструкцию:

{% for item in spisok %}
{{ item.nazvanie }}, {{ item.cena }}
{% endfor %}

Для условий (показывать/скрывать блок):

{% if flag %}
Этот текст появится, если flag = true
{% else %}
Этот текст – если false
{% endif %}

Имена переменных: только латиница, цифры, подчёркивание, начинаться с буквы или _.

⚙️ 3. Настройка шаблона в программе

Кликните на шаблон в левой панели, затем нажмите ⚙️ Настроить отображаемые имена и категории.

  • Отображаемые имена – задайте русские подписи для полей (удобно для пользователя).
    Двойной клик по ячейке «Отображаемое имя» – редактирование.
  • Тип поля – выберите text, number, date или bool. Для bool в форме появится чекбокс (удобно для условий).
  • Формат поля – для типов date и number можно выбрать один из готовых форматов или указать свой. Подробнее – ниже.
  • Категории – группируйте поля по вкладкам. Создавайте, переименовывайте, меняйте порядок перетаскиванием. Перетащите поле на категорию слева или выберите из списка.

📅 4. Форматирование дат

Для полей типа date доступны предустановленные форматы (например, DD.MM.YYYY, DD Month YYYY (рус)).

Вы также можете указать свой формат, выбрав «Свой формат...» и введя строку в стиле strftime (Python). Примеры:

Формат (что ввести)Пример вывода
%d.%m.%Y31.12.2025
%d %B %Y31 декабря 2025
%d %b %Y31 дек 2025
%Y-%m-%d2025-12-31
%d.%m.%y31.12.25

Примечание: Форматы с %B (полный месяц) и %b (короткий месяц) автоматически выводят русские названия, даже если в системе не установлена русская локаль.

🔢 5. Форматирование чисел

Для полей типа number доступны предустановки, а также «Свой формат...» с использованием Python f-строк.

Правила написания своего формата: {:,.2f} руб. означает: «два знака после запятой, разделитель тысяч – запятая, добавить текст " руб."».

В программе запятая в разделителе тысяч автоматически заменяется на пробел, а точка дробной части – на запятую (для русской нотации). Примеры:

Что ввести в «Свой формат»Пример вывода для числа 1234.56
{}1234.56
{:,.0f}1 235
{:,.2f}1 234,56
{:,.2f} руб.1 234,56 руб.
{:,.2f} ₽1 234,56 ₽ (если шрифт поддерживает символ)

Если вы хотите целое число с валютой, используйте {:,.0f} руб.. Количество знаков после запятой задаётся цифрой после точки: :.2f – два знака, :.0f – целое.

📄 6. Заполнение формы и генерация документа

После выбора шаблона справа появятся вкладки (по категориям). Заполните поля.
Для повторяющихся блоков используйте кнопки + Добавить ещё и 🗑️ Удалить.

Скачать DOCX – готовый документ.
Очистить форму – сбросить все поля.

💾 7. Автосохранение черновиков

Все изменения сохраняются автоматически (каждые 0.5 секунды). При повторном открытии шаблона данные восстанавливаются.
После нажатия «Скачать DOCX» черновик удаляется.

📦 8. Массовая генерация из CSV/Excel

Файл → Массовая генерация из CSV/Excel
Выберите файл, сопоставьте колонки с полями шаблона, нажмите «Сгенерировать ZIP».
Программа создаст архив с отдельным документом для каждой строки.

👁️ 9. Предпросмотр документа

Файл → Предпросмотр документа
Генерирует временный DOCX и открывает его в Microsoft Word (или другой программе по умолчанию).

📎 10. Помощник разметки

Файл → Помощник разметки
Три вкладки:
- Простые поля – создание тега {{ var }}.
- Повторяющиеся блоки – создание цикла {% for ... %}.
- Условные блоки – создание {% if ... %} с опциональным {% else %}.
После генерации тег копируется в буфер обмена – вставьте его в Word.

⚡ 11. Горячие клавиши

Ctrl+N – помощник разметки
Ctrl+O – выбрать папку с шаблонами
Ctrl+Shift+S – предпросмотр документа
Ctrl+Shift+G – массовая генерация

Примечание: Все имена переменных – только латиница. Кириллица в тегах не поддерживается движком Jinja2.
